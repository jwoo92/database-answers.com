Tools for the DBA
 

This Page describes some of the Tools which are available to help the DBA to do his (or her!!) job.

For example, Shell Scripts and SQL Scripts. The general trend is towards more automation of common repetitive tasks, and for a Production DBA, this means more opportunity to get involved in different areas such as Data Architecture.
